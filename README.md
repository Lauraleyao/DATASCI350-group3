# DATASCI350-group3
This is the repository for DATASCI350 final project.

# Overview
This repository contains all code, data, documentation, and outputs for our final project analyzing how water access, sanitation, education, and GDP relate to life expectancy in Sub-Saharan Africa and South Asia from 2000–2020.
The full analysis, figures, regression results, and narrative explanation are contained in the Quarto document (analysis.qmd) and its rendered output (analysis.html).

# Repository Structure
1. analysis.qmd
The main Quarto document for the project. Contains all sections: introduction, data description, heatmaps, regression analysis, scatterplots, discussion, and conclusion.
2. analysis.html
The rendered HTML output of analysis.qmd. 

# Folders
**data/**
Raw input datasets and data-pulling scripts.

Files:
pulldata.py
 Script used to pull raw indicators from WDI (via wbgapi). Produces both the CSV and SQLite database used in subsequent analysis.
qtm350_final.csv
 The cleaned, merged dataset used for statistical analysis, heatmaps, and modeling.
qtm350_final.db
 SQLite database version of the dataset, used for SQL practice and generating summary tables.
 
**documentation/**
Contains the variable codebook for transparency and reproducibility.

Files:
codebook.md – human-readable codebook describing all variables.
codebook.html – rendered HTML version.
codebook_files/ – additional files generated by Quarto.

**figures/**
All static figures produced for the project.
Include:

1. Correlation heatmap (correlation_heatmap.png)
2. Scatterplots for variables of interest over time, split by sub-Saharan Africa and South Asia (water_region.png, sanitation_region.png, life_expectancy_region.png, gdp_region.png) 
3. Scatterplots for highest 3 GDP and 3 lowest GDP countries in South Asia based on variables of interest (sa_enrollment_topbottom.png, sa_gdp_topbottom.png, sa_lifeexp_topbottom.png, sa_sanitation_topbottom.png, sa_water_topbottom.png) 
4. Scatterplots for highest 3 GDP and 3 lowest GDP countries in sub-Saharan Africa based on variables of interest (ssa_enrollment_topbottom.png, ssa_gdp_topbottom.png, ssa_lifeexp_topbottom.png, ssa_sanitation_topbottom.png, ssa_water_topbottom.png) 
5. Random forest output visualizations (Regional_Influence_on_Life_Expectancy.png, South_Asia_Country_Comparison_(Highest_to_Lowest_GDP).png, Sub-Saharan_Africa_Country_Comparison_(Highest_to_Lowest_GDP).png)
6. EntityRelationshipDiagram.png – ER diagram showing schema for the SQLite database and summary view.

**scripts/**
Python scripts used to generate analysis outputs.

Files:
correlation_heatmap.py - Computes region-level correlations and saves the heatmap figure.
linear_regression.py - Runs parallel lines and interaction models for the three predictors (water access, sanitation access, primary school enrollment). Saves model results and regression-based plots.
randomforest.py - Fits a random forest model to estimate variable importance.
scatterplots.py - Generates scatterplots comparing top vs. bottom GDP countries for each region.
sqldatasummary.py - Creates summary tables using SQL, grouped by region, and outputs the dataset used for the ER diagram.

**references.bib**
Bibliography file used in the Quarto document for automated citation formatting.

# To reproduce the analysis:
1. Run pulldata.py to download/construct raw datasets.
2. Run scripts/ to generate figures and analysis outputs.
3. Render analysis.qmd with: quarto render analysis.qmd
