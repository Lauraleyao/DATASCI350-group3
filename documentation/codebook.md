## Codebook

| Variable Name                    | Data Type | Allowed Values / Range  | Description                          |
|----------------------------------|-----------|-------------------------|--------------------------------------|
| `Country`                        | String    | Country abbreviations   | 3 letter country abbreviation        |
| `Year`                           | Integer   | 2000-2020               | Year                                 |
| `gdp_per_capita_constant_usd`    | Numeric   | ≥ 0                     | GDP per capita (constant 2015 US$)   |
| `primary_school_enrollment`      | Numeric   | 0–100+ (%)              | School enrollment, primary (% gross) |
| `basic_water_access_pct`         | Numeric   | 0–100 (%)               | People using at least basic drinking water services |
| `basic_sanitation_access_pct`    | Numeric   | 0–100 (%)               | People using at least basic sanitation services |
| `life_expectancy_birth`          | Numeric   | ≥ 0                     | Life expectancy at birth             |
| `Region`                         | String    | Region names            | Geographic region classification     |
